# 🕹️ Web Mini-Games & Playground

Hey there! Welcome to my little corner of GitHub. This repository is a collection of small web games and experiments I put together while teaching myself the fundamentals of **HTML5, CSS3, and JavaScript**. 

Instead of just watching endless tutorials, I wanted to actually build things, break them, and figure out how to fix them. None of these are meant to be production-ready masterpieces—they were just fun, "time-pass" challenges I set for myself to see if I could make the code do what I wanted.

Every single game here is entirely self-contained within its own file, making it lightweight and super easy to look through.

---

## 🏠 Game Hub

Want to play all games in one place? Check out the **`GameHub/`** folder!

A centralized hub where all games can be accessed and played without leaving the page. Simply open `GameHub/index.html` and select any game from the sidebar.

## 🎮 What's Inside?

Here is the current layout of my learning playground:

| Project Folder / File | What it is | What I practiced |
| :--- | :--- | :--- |

## 🧠 What I Learnt Along the Way

Building these tiny projects helped me wrap my head around a lot of things that didn't make sense from just reading books:
* **The DOM (Document Object Model):** Figuring out how to grab an HTML element with JavaScript and actually make it do something when a user clicks or types.
* **State Management:** Learning how to keep track of scores, active timers, and "game over" flags without the whole program crashing.
* **Layouts:** Getting real, hands-on practice with Flexbox and Grid to make sure elements actually align where they are supposed to.

## 🛠️ How to Play

You don't need to install anything complex, spin up a server, or run any build commands. 

1. Clone this repository to your laptop.
2. For individual games: Open the `Games` folder (or the `calculator` folder) and double-click any `.html` file.
3. For all games at once: Open `GameHub/index.html` to launch the centralized game hub!
